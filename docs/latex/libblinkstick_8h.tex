\subsection{Detailed Description}
Header for lib\+Blink\+Stick. 

lib\+Blink\+Stick is meant to to be a simple library for talking to a blinkstick Both the blinkstick square and blinkstick v1.\+1 are officially supported and it\textquotesingle{}s likely that other devices work as well.

lib\+Blink\+Stick depends on hidapi for interacting with the U\+SB devices, but otherwise has no other dependencies.

Included are functions to turn on and off any of the L\+E\+Ds and change their color via an \hyperlink{structrgb__color}{rgb\+\_\+color} struct. Multiple devices are supported.
